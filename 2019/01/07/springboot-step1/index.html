三. appassembler-maven-plugin生成跨平台启动脚本

执行 mvn package appassembler:assemble完成脚本的生成
优点:

  • jar包分散容易管理
  • 编译发布代码速度快
  • 可配置jvm相关启动参数
  • 日志管理
  • shell启动，停止、重启方便

四、docker-maven-plugin生成docker使用包
优点:

  • 减少Dockerfile编写
  • 增强代码一致性
  • 编译部署方便
  • 快速运维。
    缺点：
  • 编译后包比较大
  • 无法替换局部jar文件
